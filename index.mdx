      ```bash
      git clone https://github.com/ehcaw/splat.git
      cd splat
      pip install -r requirements.txt
      python setup.py install
      ```
    </Step>

      ```bash
      export API="your-groq-api-key"
      ```

      </Note>
    </Step>
    
    <Step title="Run Your First Debug">
      Use Splat to debug any Python script:
      
      ```bash
      splat squash "python3 your_script.py"
      ```
      
      For deeper analysis with dependency tracking:
      
      ```bash
      splat squash -r "python3 your_script.py"
      ```
      
      <Accordion title="Example Output">
